Berawal di SMA (tahun 2018), saya diperkenalkan dunia Pemrograman oleh teman sebangku saya. Bersama mengikuti ekstra TIK dengan julukan CLUTIK, saya mempelajari pemahaman dasar tentang hardware dan pemrograman software menggunakan bahasa Pascal. Sedikit membingungkan untuk seseorang yang sama sekali tidak memiliki latar belakang apapun tentang dunia IT.


Berbekal dari ilmu yang dimiliki, saya memberanikan diri untuk terjun secara penuh pada bidang ini. Lalu disinilah saya sekarang, mengerjakan proyek pengembangan program perangkat lunak untuk perorangan, perusahaan kecil-menengah, hingga perusahaan besar. Ketimbang merasa terbebani, saya selalu menaggap bahwa proyek yang saya kerjakan merupakan permainan problem-solving jangka panjang.


Saat ini saya lebih berfokus pada pengembangan diri untuk tidak terpaku pada zona nyaman :)
